# Commons JSON schemas

The JSON Schemas referenced by registry items and CLI output
(`https://commonsui.com/schema/registry-item.v1.json`,
`https://commonsui.com/schema/catalog.v1.json`,
`https://commonsui.com/schema/cli-output.v1.json`) are **not published yet**.

They will be published here, versioned and immutable, before the registry
contract is declared stable. Until then, treat the `$schema` URLs in registry
and CLI output as identifiers rather than resolvable documents.
