Commons is pre-1.0 and experimental. Everything here is subject to change, and
this page is meant to give a sense of direction rather than firm dates or version
numbers. Right now there is one concrete priority; more will be shared here as it
firms up.

## Up next

### Parity between the HTML and React components

Commons already ships two ways to build: a React component library, and a
framework-agnostic path built on `commons.css` classes plus the
`@21stgov/commons-js` progressive-enhancement runtime. Our current focus is
closing the remaining gaps between them.

The goal is full 1:1 parity between the `.cui-*` HTML components and their React
counterparts, in both look **and** behavior, so every component works identically
whether you reach for React or plain HTML in any stack. If you are building
without React today, the [framework-agnostic guide](/docs/without-react) is the
place to start.

## Coming soon

### Drupal and WordPress packages

Drupal and WordPress run a large share of local-government sites, and Commons
already works in both today — the [framework-agnostic guide](/docs/without-react)
shows how to load `commons.css` + `commons-js` and includes copy-paste wiring
(and downloadable starter files) for each.

The next step is dedicated, platform-native packages in their own repositories:
a Drupal module/theme and a WordPress plugin that register the Commons assets
for you and, over time, provide platform-native building blocks (Drupal
Single-Directory Components, WordPress blocks) instead of hand-written markup.
These will ship and version separately from the npm packages, and only once they
are tested against real Drupal and WordPress installs — an untested integration
is not something we want government teams to depend on.

### More to come

As additional work firms up, we will add it here rather than promise it before
we are confident in it. We would rather keep this page short and honest than pad
it with items we are not ready to stand behind.

## Share what you need

Commons is built for U.S. local governments, and input from the people using it
shapes what comes next. If there is something you need,
[open an issue or start a discussion on GitHub](https://github.com/21stgov/commons).
